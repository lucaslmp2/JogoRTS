Novo Projeto de Jogo
=====================

Descrição
---------
Este repositório contém um protótipo de jogo feito em Godot 4 (GDScript).

Requisitos
---------
- Godot Engine 4.x

Como executar
-------------
1. Abra o Godot e escolha o projeto `project.godot`.
2. Abra a cena principal `Mundo.tscn` e pressione Play.

Estrutura principal
------------------
- `project.godot` - arquivo do projeto Godot.
- `Mundo.tscn` - cena principal.
- `Global.gd` - singleton/global.
- `Unidade.tscn` / `unidade.gd` - unidade base.
- `Hud.tscn` / `hud.gd` - interface.
- `Arvore.tscn` / `arvore.gd` - recurso árvore.
- `Ouro.tscn` / `ouro.gd` - recurso ouro.
- `.gitignore` - padrões de arquivos ignorados.
